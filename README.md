# Extreme Deployment Documentation
Documentation of required steps to deploy Extreme Networks fabric at John Deere sites.

## There are 2 main deployment methods for a site.
1. Standard Deploy - Standard deploy is for smaller sites where a cutover can be done in a single outage or factory shutdown.
2. Phased Deploy - Phased deploy is for larger sites where cutover to site is impractical in a single outage and changes need to be done in phases over time while maintaining site availability.
